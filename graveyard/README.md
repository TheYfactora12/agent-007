# Experiment Graveyard

## Purpose

The graveyard stores paused, failed, replaced, or abandoned experiments.

This is not a trash folder.

It is a learning and salvage system.

The goal is to:

- preserve useful learning
- prevent repeated mistakes
- salvage reusable components
- document why a path failed
- accelerate future projects

## Required structure for each experiment

Create one markdown file per experiment.

Example:

```text
graveyard/
  replay-analysis-v1.md
  local-memory-prototype-v2.md
```

## Required fields

```md
# Experiment name

## Original goal

## Why it started

## Hypothesis

## What worked

## What failed

## Root cause of failure

## What we learned

## What should be salvaged

## What should NOT be repeated

## Possible future use cases

## Final decision

- Continue later
- Pivoted
- Paused
- Killed
```

## Important rule

A failed experiment only becomes waste if the learning is lost.
