# Training Cards

## Bottom line

Training cards are short, reusable lessons from real Agent-007 missions.

They are not doctrine.

They are not long postmortems.

They are fast operating memory.

## Purpose

Use training cards to help Agent-007 avoid repeating mistakes and reuse proven patterns.

Each card should capture:

```text
situation → bad pattern → correction → new rule → reusable example
```

Use one card per mission when there is a real lesson worth reusing.

After 5 to 10 cards:

- consolidate repeated patterns into one rule
- delete or archive duplicates
